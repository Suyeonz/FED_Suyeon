[ 피그마 컴포넌트 ]

> 만들고자 하는 요소나 그룹을 선택한 후 상단의 컴포넌트 버튼을 클릭하거나
ctrl + alt + K 단축키로 만듦
레이어에서 이동도 가능

1. 일반 그룹화와는 다른 재상용성을 위한 공용모듈화 요소
2. 원본개념이 있어서 원본을 수정하면 카피버전은 자동으로 수정
3. 원본은 채워진 다이아몬드 아이콘 / 카피버전은 비워진 다이아몬드 아이콘
4. 일반적으로 컴포넌트는 프레임 바깥쪽에 만들고 관리하는 것이 좋음
5. 제작된 컴포넌트는 Assets 메뉴에 있음 또는 상단 메뉴의 Resources 버튼 아래에서도 컴포넌트를 확인 가능
6. 컴포넌트 내부에 컴포넌트를 묶어서 작업할 경우 내부 컴포넌트를 variant(변형)라고 부름
    > 단어의 뜻을 볼 때, 기본형으로부터 변경 시에 사용
